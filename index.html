這一頁做什麼:記錄每一筆收支,是最常用的頁面。

快速記一筆(10 秒)
① 輸入金額 → ② 日期預設今天(可一鍵切昨天)→ ③ 點類別按鈕,點下即完成記錄,不用按儲存。
類別按鈕會依使用頻率自動排前面。信用卡消費不用手記,月底統一匯入。

分帳(和別人分攤時)

  1. 記帳時打開「與他人分帳」開關
  2. 選方向:我代墊(我先付全額,對方之後還我)或對方代墊(對方先付,我之後還他)
  3. 填帳單總金額與對方分攤金額 → 記錄只計入「我自己的花費」
  4. 對方欠的/我欠的,自動進入「分帳總覽」追蹤

批次匯入信用卡明細(每月一次)

  1. 準備 CSV,格式:交易日期,消費說明,金額(負數=退款)
  2. 選擇檔案 → 進入預覽,先勾選要匯入的列
  3. 勾多筆 → 批次設定類別(可分批設不同類別);需要分帳的勾選後批次分帳
  4. 標示「疑似重複」的列(和既有記錄日期+金額+說明相同)保持不勾——上月已匯入的消費會被自動攔下
  5. 按確認匯入才寫入

分帳總覽(追蹤欠款)

  • 按對象顯示淨額:綠色=對方欠我,紅色=我欠對方
  • 收到還款 → 點對象 → 輸入金額(可部分還款)→ 結清
  • 結清不會產生收入/支出:你的花費在記帳當下已定案,還款只是銷掉欠款,損益表不變
  • 不綁消費的欠款(如純代墊現金)用「新增欠款」記

這一頁做什麼:看每個月的收入、支出與結餘。

  • 支出金額一律以「我的實際負擔」計算:分帳過的消費只算自己的份
  • 卡費繳納、分帳還款都不會出現在這裡(避免重複計算),數字即真實消費水準
  • 搭配記帳頁的本月支出比例,檢視各類別占比

這一頁做什麼:追蹤每月要繳的帳單,繳了沒、繳多少。

建立帳單(一次設定)

  • 固定金額(管理費、電話費):設名稱、金額、每月繳費日;可開「繳費時同步記支出」
  • 浮動金額(信用卡費):設名稱、繳費日;金額每月出帳後才填

每月操作

  1. 每月自動產生本期帳單;首頁頂部會提醒未繳與逾期
  2. 信用卡出帳後 → 填入實際應繳金額
  3. 繳費後 → 按繳費輸入金額(預設全額,可部分繳清)
  4. 部分繳清的餘額自動滾入下期並顯示警示

注意:卡費繳納不記入支出(刷卡時已逐筆記過);管理費等若開啟同步記支出,繳費時會自動產生一筆支出記錄。

這一頁做什麼:記錄股票/ETF 持股與損益。

  • 手動新增:名稱、股數、成本價、現價
  • 快速更新:券商 App 庫存截圖交給 Claude 轉成備份檔 → 設定頁匯入合併,不用逐筆改
  • 可保存庫存截圖作為對照

這一頁做什麼:資料備份、類別管理、使用說明。

備份(最重要)

  • 所有資料只存在這台裝置的瀏覽器,不會自動上傳
  • 定期「匯出並分享備份檔」存到 iCloud;超過 7 天未備份會提醒
  • 換機/誤刪 → 匯入備份檔合併還原(不覆蓋、不重複)

建議的月度節奏
平日快速記現金消費 → 月底:匯入信用卡 CSV 並批次分類分帳 → 帳單頁填卡費、繳費打勾 → 分帳總覽結清已收還款 → 匯出備份。

備份與還原

資料只存在這個瀏覽器裡，手機和電腦不會自動同步。匯出備份檔後傳到另一台裝置匯入，就能把兩邊的紀錄合併在一起(不會覆蓋、也不會重複)。

清除資料

這會刪除這個瀏覽器裡所有記帳、投資、截圖與自訂類別資料，且無法復原。建議先匯出備份再清除。
